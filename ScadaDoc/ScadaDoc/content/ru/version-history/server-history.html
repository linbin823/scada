История приложения Сервер

5.1.0.0 (В разработке)
- В API модулей добавлена отправка команд через каналы управления
- Исправлена ошибка получения текущего среза из серверных модулей

5.0.2.0 (27.06.2017)
- Функции SetVal() и SetStat() возвращают double и int соответственно
- Добавлена функция SetData()
- Добавлена загрузка фраз для локализации модулей в службе Сервера

5.0.1.0 (04.03.2017)
- Возможность прямого получения и передачи данных Серверу из модуля
- Функции SetVal() и SetStat() для установки данных входных каналов из формул базы конфигурации
- Установка архивного статуса при записи архивных срезов для всех типов входных каналов
- Возможность установки значений дорасчётных каналов клиентом
- Событие нормализации не возникает после неопределённого состояния
- Поддержка номеров версий модулей

5.0.0.0 (31.10.2016)
- Работа нескольких экземпляров службы
